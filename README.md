Xposed is a python based tool used for Email and password breach lookup to check for leaked credentials and exposed passwords.
https://api.xposedornot.com/

Installation Dependencies: 
python 3.11
sudo apt install git
pip3 install -r requirements.txt


Details:

 Email verification in multiple data breaches

 Risk level detection (High, Medium, Low)

 Detailed information for each gap found

 Exposed password statistics

 Colourful and easy-to-read interface

 Industry verification support
